SetGiftWrappingOptionsOnOrder

Set the GiftWrapping options on an Order. There are 4 options: - Wrap the complete order: set `WrapOrder` to `true` - Wrap the complete order individually: set `WrapIndividually` to `true` - Wrap parts of the order: set `GiftWraps` with a set of wraps - Wrap nothing: set everything to `null` A `GiftWrap` consists out of an optional `Message` (could be printed on a card), an optional `GreetingCardProductID` and a set of `WrapLines`. A `WrapLine` has an `OrderLineID` and a `Quantity`. If a `GreetingCardProductID` and/or `Message` is desired when the order should be completely wrapped, or without any giftwrapping at all, set these properties on root level. In all other cases they should be set on the `GiftWraps`.
