GetReplenishmentProposal

Get a `ReplenishmentProposal`. * To get the possible sources of this proposal, use the 'GetReplenishmentProposalSources`. * To get the targets of this proposal, use the 'GetReplenishmentProposalTargets`. * To get the stock labels of this proposal, use the 'GetReplenishmentProposalStockLabels`. * To get the results of this proposal, use the 'GetReplenishmentProposalResults`.
