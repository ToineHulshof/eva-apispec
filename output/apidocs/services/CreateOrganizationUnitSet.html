CreateOrganizationUnitSet

Creates a new `OrganizationUnitSet`. Optionally a definition can be provided, which defines a set of filters. Any `OrganizationUnit` that matches the filters are automatically part of the set. If a definition is provided without any filters, it's ignored. The ScopeID refers to an `OrganizationUnitSetScope` and can be used to create collections of sets for specific purposes, such as discounts.
