ListAvailableUserTasks

List the available UserTasks for the current user. The response has a `Aggregations` property. This property contains the available filters for this set of UserTasks. To apply such a filter it should added in the `Filters` property on the RequestMessage. Example response: ``` { ... , "Aggregations": { "ZonedCycleCountPreCount": { "CycleCountZoneID": [ { "Name": "Shelves", "Value": 10, "Count": 1 }, { "Name": "Stockroom", "Value": 12, "Count": 6 }, { "Name": "Store", "Value": 13, "Count": 4 } ] } } } } ``` To apply a matching filter: ``` { "Filters": { "ZonedCycleCountPreCount": { "CycleCountZoneID": [10] } } } ```
