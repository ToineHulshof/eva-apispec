GetStockAvailabilityEstimateForOrder

Calculates the stock availability for the provided OrderLines that are part of the provided Order. The return value of this service is a list that contains, per OrderLine, whether or not it can be fulfilled (`CanBeFulfilled`) on the `RequestedDate`. When an OrderLine doesn't already have a RequestedDate, a RequestedDate of tomorrow is used. Additionally, a list of AlternativeDates is returned per OrderLine with dates on which it is expected that the OrderLine can be fulfilled. When OrderLineIDs is left empty, this service will return an estimate for all unshipped lines of the order. Doesn't work for orders with more than 25 lines.
