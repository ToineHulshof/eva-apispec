Barcode

Generate a barcode image. The `Format` is one of; - QR = 0 - EAN = 1 - Code39 = 2 - Code128 = 3 :lock: This is the safer variant to our `/barcode` endpoint, because the data moves _inside_ the POST body instead of through GET query params :bulb: You can use the returned `Data` in combination with `MimeType` to render the image directly, for example; ``` <img src='data:{MimeType};base64,{Data}' /> ```
