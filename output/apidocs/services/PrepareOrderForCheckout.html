PrepareOrderForCheckout

Prepare the order to go into the checkout If the order still has status ShoppingCart, update it to Order and: - Validate lineactiontypes (customer are not allowed to use ShipLine p/a) - Set the SoldBy on the lines to the current user - Set the addresses of the customer on the order, if not already set - (Re)calculate the taxes - Recalculate the order: discounts, shippingcosts, giftwrappingcosts etc.
