ProcessStockMutationFile

Processes a (delta)stock mutation file. Expected format of the file is JSON with the following markup: ``` [ { "ProductID": [Optional, int], "ProductBackendID": [Optional, string], "QuantityOnHand": [Required, int, value >= 0] }, ... ] ``` Mutation will only be processed for valid ProductIDs or ProductBackendIDs. In case ProductID and ProductBackendID are provided, ProductBackendID will be selected as the search parameter.
