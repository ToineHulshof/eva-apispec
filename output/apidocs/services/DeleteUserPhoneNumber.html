DeleteUserPhoneNumber

Delete a user's phone number based on the provided ID. If you're trying to delete a user's primary phone number and that user has more than two phone numbers currently, this service will return a `CannotDeletePrimaryPhoneNumber` error, because it cannot determine what number should now become the user's primary phone number. If the user has two numbers, the other number will become the primary. This service only allows the number's own user or an employee to delete it.
